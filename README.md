### 01 Frontend a.k.a client

#### Technology used 
- ReactJS
- TailwindCSS
- Vite
- Zustand
- React-Toastify
- Flowbite
- lucide-react

#### How to run frontend?
1. Check NodeJs Installation
```
node --version
```

2. Check npm Installation
```
npm --version
```

3. Clone repository from github
```
git clone https://github.com/VarunSharma3520/bodhika.git
```

```
cd bodhika/client
```

4. Run npm install
```
npm install
```

5. Run npm start
```
npm start
```
